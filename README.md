# shell-webapp
Using shell only to build a web application, based on netcat. 

## Deploy 
```bash
chmod +x dispatcher.sh handler/* render/* util/*
```

## Run
To run the server:
```bash
./dispatcher.sh
```

To run a client, you can open a browser and just go `http://localhost`.  
Or you can `curl localhost`
